UI画像フォルダ
=============

このフォルダにはWebサイトのUI要素で使用する画像を配置します。

推奨ファイル:
- logo.png: サイトロゴ
- favicon.ico: ファビコン
- background.jpg: 背景画像（必要に応じて）

icons/ サブフォルダ:
- home.svg: ホームアイコン
- search.svg: 検索アイコン
- user.svg: ユーザーアイコン
- star.svg: 評価星アイコン
- location.svg: 位置アイコン

形式: PNG, SVG推奨（アイコンはSVG、写真はJPEG/PNG）
最適化: Web用に圧縮済みの画像を使用
